# リポジトリ境界

*最終更新: April 9, 2026 at 12:49 AM PDT*

## 公開リポジトリ

公開する GitHub リポジトリには、再利用可能なプロジェクト素材だけを含めます。

- ソースコード
- migration
- 汎用設定テンプレート
- 英語のプロジェクト情報
- 英語と日本語の利用者向けドキュメント

追跡されるコードコメントは、すべて英語で記述します。

## ローカル専用の内容

以下はバージョン管理に含めないでください。

- 個人用のデプロイメモ
- 非公開の企画書や草稿
- ローカルのスクリーンショット
- 一時的な作業ファイル
- 本番 secrets
- 個人ドメイン、アカウント識別子、バケット名
- AI 支援ツールのワークスペース、ログ、セッション状態

このリポジトリでは `local/`、`private/`、`.cursor/`、`.claude/`、`.codex/`、`.windsurf/` などのローカル専用ディレクトリを無視します。

## ドキュメント配置

推奨構成は次のとおりです。

- `README.md`: 英語のみのリポジトリエントリ
- `docs/en/`: 英語ドキュメント
- `docs/ja/`: 日本語ドキュメント
- `local/`: 任意のローカル専用メモ。Git では無視されます

## 設定ポリシー

- `wrangler.toml.template` にはプレースホルダーのみを置きます
- 実際の `wrangler.toml` はローカル専用にします
- 機密値は Wrangler secrets を使います
- アカウント固有の route、本番 origin、アクセスキー、セッション secret はコミットしません
